
[Esc] edit/command mode

Edit mode:

   [Control+T] nou tab
   [Alt+left]/[Alt+right] tabs
   [Control+W] tanca tab

   [Control+R] engega/atura la grabació de macro,
   [Control+P] la playa,
   [Control+E] la playa repetint fins arribar a EOF (oju als bucles infinits),
   
   [Control+O] open file
   [Control+S] save file
   
   [Control+Q] quit
   [Control+Z] undo
   [Control+Y] redo
   [Control+X] cut
   [Control+C], [Control+Insert] copy
   [Control+P], [Shift+Insert] paste
   
   [Selection+Tab] tabula bloc
   [Selection+Shift+Tab] destabula bloc

   [Control-Left] left word
   [Control-Right] right word
   [Control+Up] scroll up
   [Control+Down] scroll down

 Command mode:

   f[text]      - cerca endavant (case sensitive)
   F[text]      - cerca enrera (case sensitive)
   @f[regexp]   - cerca endavant per regexp
   #[numfila]   - go to # fila
   u            - sel.lecció.toLowerCase()
   U            - sel.lecció.toUpperCase()
   ![cmd]       - inserta la sortida d'un comando de línia
   [            - push cursor position marker
   ]            - pop cursor position marker
